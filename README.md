# Physical AI and Humanoid Robotics: From Embodiment to Real-World Intelligence

This is a Living Book on Physical AI and Humanoid Robotics with an embedded RAG chatbot.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Project Structure

The book is organized into weekly modules:

- Weeks 1-2: Foundations
- Weeks 3-5: ROS2
- Weeks 6-7: Simulation
- Weeks 8-10: NVIDIA Isaac
- Weeks 11-12: Humanoid Development
- Week 13: Conversational Robotics

## Deployment

This site is configured for deployment on GitHub Pages. To build and deploy:

```bash
npm run build
```

The built site will be available in the `build` directory.
